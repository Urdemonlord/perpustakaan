# Sistem Informasi Perpustakaan

Aplikasi manajemen perpustakaan berbasis Laravel untuk mengelola peminjaman buku, anggota, dan denda.

## Fitur
- Manajemen Buku
- Manajemen Anggota
- Peminjaman & Pengembalian
- Perhitungan Denda
- Multi-user (Admin & User)

## Teknologi
- Laravel 10
- PostgreSQL
- Bootstrap 5
- DataTables
- SweetAlert2
